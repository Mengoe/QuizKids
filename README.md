# 개발 개요

수업에서 배운 Enact를 활용하여 WebOS기반의 미디어 웹앱 개발

# 개발 내용

사용자가 영상을 보면서 관련 퀴즈 문제를 출제하거나, 남들이 올린 퀴즈를 interactive하게 상호 공유하면서 즐길 수 있는 웹앱

# 개발 계획

- 1주차: 개발환경 세팅 및 설계
- 2~3주차: 핵심 기능 개발
- 4주차: 디버깅 및 테스트 코드 작성
- 5주차: 개발 문서 작성 및 추가 기능 구현


# 기술 스택

- F/E: Enact
- B/E: Node.js, Express
- DB: MySQL
- Documentation: Swagger, Notion
- UI/UX: Figma
- Version Control: Github

# 역할 배분

- DB, B/E: 서윤혁, 정재훈
- F/E, UI/UX: 강상원, 고동헌
- 기획: 강상원, 정재훈, 고동헌, 서윤혁
